# 01. プロジェクト概要 (Project Overview)

## 📌 プロジェクト概要
本プロジェクトは、カメラとQRコードを用いて、従業員の出退勤を非接触で記録・管理するWebアプリケーションシステムです。
「非接触」「UIレス」「ローカル完結」をキーワードに、小規模なオフィスや店舗での実運用を想定して設計されています。

## 💡 開発の背景と目的

### 1. 解決したい課題
* **紙ベース・タイムカード運用の非効率性**: 集計作業の手間や物理的な保管コストの削減。
* **二重打刻・誤打刻の発生**: 従業員の操作ミスによるデータ修正コストの削減。
* **導入ハードルの低減**: クラウド契約不要で、PC1台（ローカル環境）と内蔵カメラのみで即座に運用開始できる仕組み。

### 2. システムの目的
本システムは、PC内蔵カメラとQRコードを活用し、正確かつスムーズな打刻体験を提供することを目的とします。
技術的には、**FlaskによるWebアプリケーション構築**、**OpenCVによるリアルタイム画像処理**、**SQLiteによるデータ永続化**を組み合わせた実践的な成果物として開発しました。

## 🎯 ターゲット
* 小規模事業所、店舗、イベント会場など。
* 専用の打刻端末（iPad等）を導入せずに、既存のPCで運用を始めたい現場。
